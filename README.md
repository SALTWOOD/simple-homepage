# 简介
这是我自己写的纯 HTML 个人主页，旨在展示自己的一些信息，同时为我的域名的根域放些东西以免直接通到博客。

# 如何使用
1. 克隆或下载本仓库到本地。
2. 根据示例文件填充你自己的 `index.html` 文件。
3. 使用如 Nginx、Apache 之类的服务器软件将整个项目部署到你的服务器上。
4. 打开浏览器访问你的站点，就可以看到你的个人主页了。
